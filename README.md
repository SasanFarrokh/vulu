Validation Plugin for [Vue 3](https://github.com/vuejs/vue-next)

## Getting Started

Install vulu package with:
```bash
$: npm install --save-dev vulu
# or yarn
$: yarn add -D vulu
# or pnpm
$: pnpm add -D vulu
```
